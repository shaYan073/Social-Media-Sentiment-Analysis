# Social Media Sentiment Analysis

A Machine Learning based sentiment analysis web application developed using Streamlit.

## Features

- Basic Text Cleaning
- Advanced Context Enhancement
- TF-IDF Feature Extraction
- Chi-Square Feature Selection
- Linear SVC Classification

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- NLTK
- Joblib
